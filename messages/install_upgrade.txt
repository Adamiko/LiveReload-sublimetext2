Changelog:

Please be sure to restart Sublime Text 2 to start using this new version. For more information or how to use this plugin visit https://github.com/dz0ny/LiveReload-sublimetext2/blob/master/README.md 

New Features
 - Support v2 of protocol
 - SublimeText3 support
 - Plug-in API(you can write your own plug-ins)
 - Plug-ins:
  * Simple reload, demonstrates simple reloading
  * Markdown preview, demonstrates new JS API plug-ins, with two-way communication
  * Build plug-in, demonstrates simple callback operation via native call
  * Chrome remote CSS edit plug-in(You can edit CSS in Google Chrome and it gets reflected back in Sublime Text)
